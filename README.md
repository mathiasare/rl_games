# RL_Games

## Author: Mathias Are

Reinforcement learning algorithm implementations on OpenAI Gymnasium games and simulations.

Main idea was to progressively implement algorithms from simpler Q-learning to Deep Q architectures such as Actor-Critic and DDQL. Planning to add A2C and PPO next.

## Setup

1. Run `uv sync` in Terminal to sync project and install dependencies.

1. Run `uv run main.py` to see agent playing games.

1. Run the **rl_games.ipynb** notebook to train the models yourself.
